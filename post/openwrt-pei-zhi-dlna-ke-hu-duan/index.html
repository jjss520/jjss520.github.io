openwrt配置DLNA客户端

2026-02-05

. 安装 USB 音频驱动
更新软件包列表并安装 USB 音频内核模块：

opkg update
opkg install kmod-usb-audio #不是USB声卡，不用装

目的：确保系统能够识别 USB 音频设备。

  1. 安装播放组件
    安装 MPD、UPnP/DLNA 客户端和 ALSA 工具：

opkg install mpd-mini upmpdcli alsa-utils
目的：安装音乐播放守护进程 (mpd-mini)、UPnP/DLNA 客户端 (upmpdcli) 和音频配置工具 (alsa-utils)。

  1. 配置音频输出

3.1 获取声卡的设备值
检查声卡设备（假设只有一个声卡，通常为 hw:0,0）：

aplay -L
目的：列出可用声卡，确认设备名称和编号（如 hw:0,0 或 hw:CARD=AUDIO,DEV=0）。

3.2 获取声卡的硬件音量控制参数
查看声卡的混音控制（mixer control）和索引（mixer index），-c 0 表示第一个声卡（多个声卡依次为 1、2 等）：

amixer -c 0 scontrols
目的：获取硬件音量控制参数（如 PCM）和索引值（通常为 0）。

3.3 生成 MPD 配置文件 /etc/mpd.conf
根据声卡支持情况，生成配置文件：

cat << EOF > /etc/mpd.conf
audio_output {
type "alsa"
name "My ALSA Device"
device "hw:CARD=PCH,DEV=0"
mixer_type "hardware"
mixer_device "hw:CARD=PCH"
mixer_control "PCM"
mixer_index "0"
}
4.修改MPD运行用户

sed -i 's#USER="mpd"#USER="root"#' /etc/init.d/mpd
sed -i 's#GROUP="mpd"#GROUP="root"#' /etc/init.d/mpd
5.重启mpd（报错不影响使用）

service mpd restart
6.设置upmpdcli的名称

uci set upmpdcli.lan.friendly_name=OpenWrt
uci commit upmpdcli
service upmpdcli restart
